## Configuration System

This survey now supports a flexible configuration system that allows administrators to:

- **Enable/Disable Questions**: Turn individual questions on or off
- **Reorder Questions**: Customize the question order within sections
- **Multiple Configurations**: Create different survey versions for different audiences
- **Language Control**: Enable/disable specific languages
- **Required vs Optional**: Mark questions as required or optional
- **Preview Mode**: See exactly what respondents will see before activation

### Use Cases
1. **Pulse Surveys**: Short 5-10 question check-ins
2. **Department-Specific**: Different questions for different departments
3. **Regional Variations**: Different questions for NA vs Europe
4. **Onboarding Surveys**: New employee focused questions
5. **Annual vs Quarterly**: Comprehensive vs brief surveys

### Managing Configurations
Access the Survey Configuration admin panel at `/admin/survey-config` to create and manage survey configurations.
